## SEO Framework

### Keyword Research Approach

**Primary Keywords (high intent):**
- "[country] esim [market]" (e.g., "japan esim australia")
- "esim for [country]"
- "[country] travel sim card"

**Secondary Keywords:**
- "[carrier] roaming [country]" (e.g., "telstra roaming japan")
- "best esim for [country]"
- "[country] data plan for tourists"

**Long-tail Keywords:**
- "how much data do i need for [country]"
- "does esim work in [country]"
- "[country] airport sim card vs esim"

### On-Page SEO Checklist

- [ ] Primary keyword in H1 (title)
- [ ] Primary keyword in first 100 words
- [ ] Primary keyword in URL slug
- [ ] Primary keyword in meta description
- [ ] Secondary keywords in H2 headings
- [ ] Internal links to relevant pages (2-5)
- [ ] External links to authoritative sources (1-3)
- [ ] Image alt text with keywords
- [ ] 1,500-2,500 words for pillar content
- [ ] 800-1,200 words for supporting content

## Writing Guidelines

### Voice for Blog Content

- More educational than sales pages
- Authoritative but approachable
- Use "you" to address reader directly
- Share specific, actionable advice
- Back claims with data when possible

### Formatting Best Practices

- Short paragraphs (2-4 sentences)
- Bullet points for lists
- Bold key terms and takeaways
- Use tables for comparisons
- Include a table of contents for 1,500+ word posts
- Break up text with H2/H3 every 200-300 words

### Trust Elements

- Cite official carrier pricing
- Reference real user experiences
- Include "last updated" date
- Link to official sources
- Mention money-back guarantee
